______________________________________________________________________________________
                                        -DOCKER-
______________________________________________________________________________________

####-Побудова: (Перед білдом очистити існуючий контейнер)

$ docker build -t НАЗВА КОНТЕЙНЕРА . 

####-Запуск:

$ docker run -d -p 80:80 --name=НАЗВА-container НАЗВА    - наприклад (docker-app)

####-(-d) виконання контейнера в фоновому режимі
####-(-p) відкриття порту на локальнйі машині і його перенаправлення

--------------------------------------------------------------------------------------

####-Список образів:

$ docker images -a

####-Видалення образів:

$ docker rmi Image Image

--------------------------------------------------------------------------------------

####-Список контейнерів:

$ docker ps -a

####-Видалення контейнерів:

$ docker rm ID_or_Name ID_or_Namе

####-(docker rm -f) - видалення контейнера без зупинки.

______________________________________________________________________________________
                                    -DOCKER COMPOSE-
______________________________________________________________________________________

####-Побудова:

$ docker-compose up -d --build

####-Cтатус

$ docker-compose ps 

####-Вимкнення контейнерів

$ docker-compose down
